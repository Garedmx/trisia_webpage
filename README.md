# Trisia Webpage

Página web publicitaria de Trisia - Soluciones innovadoras para tu negocio.

## Estructura del Proyecto

```
trisia_webpage/
├── index.html      # Página principal
├── css/
│   └── styles.css  # Estilos CSS
├── README.md
└── LICENSE
```

## Características

- Diseño responsive para dispositivos móviles y escritorio
- Navegación suave entre secciones
- Secciones: Inicio, Servicios, Nosotros, Contacto
- Estilos modernos con gradientes y animaciones

## Uso

Para visualizar la página localmente, puedes abrir `index.html` directamente en tu navegador o usar un servidor local:

```bash
python3 -m http.server 8080
```

Luego visita `http://localhost:8080` en tu navegador.
